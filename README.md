# Go React Starter

Use Go and React without missing out on hot module reloading while in development.

### Dev
```shell
# One terminal pane
$ cd server
$ go build && ./server

# Another terminal pane
$ npm start
```

Then visit `localhost:3334`.

### Prod
```
$ npm build
$ cd server
$ go build && ./server
```

Then visit `localhost:3333`
